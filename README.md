# BWL Planspiel Factory

Digitale Umsetzung des Planspiels "Factory" als interaktive Web-App.

---

## Starten (Windows)

### Schritt 1 – Python installieren (nur beim ersten Mal)

1. Gehe auf **https://www.python.org/downloads/**
2. Lade die neueste Version herunter und installiere sie
3. **Wichtig:** Beim Installieren das Haekchen bei **"Add Python to PATH"** setzen

### Schritt 2 – Anwendung starten

1. ZIP-Datei entpacken
2. Doppelklick auf **`starten.bat`**
3. Ein schwarzes Fenster oeffnet sich kurz (installiert Abhaengigkeiten beim ersten Start)
4. Der Browser oeffnet sich automatisch mit der Anwendung

> Das schwarze Fenster muss waehrend der Nutzung offen bleiben. Schliessen beendet die App.

---

## Haufige Probleme

**"Python ist nicht installiert"**
→ Schritt 1 ausfuehren und darauf achten, dass "Add Python to PATH" angehakt ist

**Browser oeffnet sich nicht automatisch**
→ Manuell aufrufen: `http://localhost:8501`

**App startet nicht / Fehlermeldung im schwarzen Fenster**
→ Screenshot des Fehlers an den Ersteller schicken
